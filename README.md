# ArduinoToyCar

# Transmitter - это скетч для пульта.

Джойстик оси Y идет в пин A0, оси X - в пин A1. Кнопка от джойстика присоединяется на D2, от второго (если надо) - на D3. Радиомодуль подключается как на картинке: https://github.com/ghB111/ArduinoToyCar/blob/master/7ozG6MuaLmE.jpg.

# Receiver - это скетч для приемника в машинке.

Сервопривод подключается к D6. Радиомодуль подключается как на картинке. Двигатель: ENA с модуля идет в D3 на ардуино, IN1 - D2, IN2 - D4. 

# nRF24L01 tests - это папка со скетчами для тестов модулей, если вдруг понадобится.

Среди них самый полезный это nrf_listen_air, который показывает, какие частоты "захламлены", а на каких сигнал будет чистый.

# Заметки
Сервопривод на 30 влево, 30 вправо.
Можно поменять тип данных у массива данных, может, будет быстрее.
Контроль подключения: если нет пакетов X секунд - DATA.reset();
Можно тип у массива поменять на uint8.
