# ArduinoToyCar

# Transmitter - это скетч для пульта.

Джойстик оси X идет в пин A1, оси Y - в пин A2. Кнопка от джойстика присоединяется на D1, от второго (если надо) - на D2. Радиомодуль подключается как на картинке: https://github.com/ghB111/ArduinoToyCar/blob/master/7ozG6MuaLmE.jpg.

# Receiver - это скетч для приемника в машинке.

# nRF24L01 tests - это папка со скетчами для тестов модулей, если вдруг понадобится.

Среди них самый полезный это nrf_listen_air, который показывает, какие частоты "захламлены", а на каких сигнал будет чистый.
